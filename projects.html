Cayley 1: Autonomous Quad Rotor Helicopter
Robotics C/C++ PCB PID Complete

Cayley 1 is a home made, low cost quad rotor helicopter, capable of autonomous flight. It is controlled wireless by an operator using Zigbee wireless comms. It's self stabilising, using an IMU to balance itself in the air and features a servo arm mounted ultrasonic sensor for distance measurements of its surroundings. I wrote all of the flight control software and designed and built the aircraft's PCB myself.

ALBot: Musical toy with FPGA based machine vision
VHDL C Machine Vision Complete

ALBot is a moving, musical toy that responds to colours and lines in it environment. An FPGA was used to produce rudimentary machine vision, able to detect when a certain colour is present in the frame and what colour that is. It can also follow black lines on the floor. Using its colour and line detection, ALBot will move around and produce music. ALBot also has stuffed plush exterior to make it a bit more child friendly and awesome looking!

Voting Webservice: Java w/ Akka PoC backend
Java Akka InfluxDB PoC

This was a proof of concept (PoC) for the backend of my voting webservice. It is written in Java and leveraged the excellent Akka actor library, making the whole thing upward and outwardly scalable, with each HTTP request making a new actor handler. I dropped in the second iteration for the Go language instead, but I will definitely come back and use Akka for something again!
